# Layout 布局

# Layout

@/views/layout

本项目布局分为tapbar,sidebar,tags-view,content4个模块。 统一都放在 @/views/layout 文件夹中。

index.vue 式布局的入口文件：

<template>
  <div class="yus-layout">
    <el-container class="yus-container">
      <topbar class="yus-header"></topbar>
      <el-container class="yus-body">
        <div class="yus-menu">
          <sidebar></sidebar>
        </div>
        <el-main class="yus-main">
          <tagsView></tagsView>
          <div class="yus-content">
            <keep-alive :include="cachedViews">
              <router-view></router-view>
            </keep-alive>
          </div>
        </el-main>
      </el-container>
    </el-container>
  </div>
</template>
// ...
