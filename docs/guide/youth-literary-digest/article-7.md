# 揭秘 JS 之 你的Promise

古人云:`"君子一诺千金"`,这种“承诺将来会执行”的对象在`JavaScript`中称为`Promise`对象

在`JavaScript`世界里，只有单线程，没有多线程。所以很多场景都需要使用`异步`来解决，`Promise` 是一种异步编程的解决方案。
